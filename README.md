# Zm

Zm is command line replacer.

## Usage

```
$ zm -h
Zm: v0.1.0

usage: zm [OPTIONS] -- [COMMANDLINE]...

options:
  -f, --file <FILE>                path of JSON configuration file
  --show_keyword_with <DELIMITOR>  show keyword name with given delimitor like 'keyword=value'
  -h, --help                       print help
```
